# 🌌 CosmoLens

CosmoLens is an advanced astronomy simulation system with two modules:

## ⭐ Star Atlas (Main System)

Run:

```
python cosmolens.py
```

Features:

* Hipparcos star catalog
* Constellations
* Planets (Skyfield)
* Interactive sky navigation

---

## 🛰 Satellite Tracker

Run:

```
python main.py
```

Features:

* Real-time satellite tracking
* TLE data (gp.php)
* Orbit visualization on Earth map

---

## 📦 Requirements

```
pip install pygame skyfield numpy pandas requests
```

---

## 📌 Notes

* `gp.php` contains satellite TLE data (not PHP code)
* `de421.bsp` auto-downloads if missing
